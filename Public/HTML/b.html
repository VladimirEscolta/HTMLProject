Тег <b>

Описание

Устанавливает жирное начертание шрифта. Допустимо использовать этот тег совместно с другими тегами, которые определяют начертание текста.

Атрибуты

Пример <b>Текст</b> История о том, как возле столовой появились загадочные розовые следы с шестью пальцами, и почему это случилось. Аналог CSS - font-weight
