Тег <var>

Описание

Тег <var> используется для выделения переменных компьютерных программ. Браузеры обычно помечают текст в контейнере <var> курсивным начертанием.

Атрибуты

type <var>Текст<var> Произвольный текст. Будем рассматривать переменную t, которая используется для чего-нибудь. Переменная t будет выделена в тексте. ...
