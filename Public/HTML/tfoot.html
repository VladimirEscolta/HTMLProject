Тег <tfoot>

Описание

Элемент <tfoot> предназначен для хранения одной или нескольких строк, которые представлены внизу таблицы. Хотя тег <tfoot> в исходном коде должен быть определен до тега <tbody>, браузеры отображают его в самом низу таблицы. В пределах таблицы разрешается использовать только один элемент <tfoot>.

Атрибуты

align <tfoot align="left | center | right | justify">...</tfoot
Объединенная ячейка в TFOOT
Ячейка 1, расположенная в TBODY Ячейка 2, расположенная в TBODY

Задает выравнивание по горизонтали для всех ячеек в пределах <tfoot>. Если требуется применить разное выравнивание для каждой ячейки, воспользуйтесь стилями или используйте атрибут align для тега <td> или <th>.

left
Выравнивание содержимого ячейки по левому краю.
center
Выравнивание по центру.
right
Выравнивание по правому краю.
justify
Выравнивание по ширине (одновременно по правому и левому краю).
bgcolor <tfoot bgcolor="цвет">...</tfoot
Объединенная ячейка в TFOOT
Ячейка 1, расположенная в TBODY Ячейка 2, расположенная в TBODY

Устанавливает цвет фона ячеек, расположенных в контейнере <tfoot>.
char <tfoot char="c" align="char">...</tfoot
189767,5 млн
1939108,4 млн
1989147,0 млн
2007142,2 млн
2008142,0 млн
2009141,9 млн
2010141,93 млн

Атрибут char заставляет выравниваться содержимое ячеек внутри <tfoot> по заданному символу. Эта возможность может пригодиться для выравнивания колонки чисел по точке или запятой. Работает только в том случае, если значение атрибута align установлено как char.
charoff <tfoot char="c" align="char" charoff="число">...</tfoot
189767,5 млн
1939108,4 млн
1989147,0 млн
2007142,2 млн
2008142,0 млн
2009141,9 млн
2010141,93 млн

Если атрибут align установлен как char, а для атрибута char в качестве значения задан символ, то с помощью атрибута charoff можно смещать содержимое ячеек влево или вправо относительно заданного символа. Так, charoff="3" означает, что браузер должен сместить содержимое ячейки на три символа вправо, а charoff="-3" — сместить на три символа влево.
valign <tfoot valign="top | middle | bottom | baseline">...</tfoot
tfoot 1 tfoot 2
Ячейка 1 Ячейка 2
Ячейка 3 Ячейка 4
Ячейка 5 Ячейка 6

Устанавливает вертикальное выравнивание содержимого ячеек в контейнере <tfoot>. По умолчанию контент ячейки располагается по ее вертикали в центре.

top
Выравнивание содержимого ячеек по верхнему краю строки.
middle
Выравнивание по середине.
bottom
Выравнивание по нижнему краю.
baseline
Выравнивание по базовой линии, при этом происходит привязка содержимого ячеек строки к одной линии.
