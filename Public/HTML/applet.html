Тег <applet>

Описание

Тег <applet> предназначен для вставки на страницу апплетов — небольших программ, написанных на языке Java. Этот тег является устаревшим, взамен необходимо использовать более гибкий тег 'object'. Между открывающим и закрывающим тегом можно добавить текст, который будет отображаться в браузере, если он не поддерживает апплеты. В противном случае текст не выводится.

Атрибуты

align <applet align="bottom | left | middle | right | top">

Текущее время: Ваш браузер не поддерживает Java-апплеты.

Задает выравнивание апплета относительно близлежащих элементов и текста.
alt <applet code="URL" alt="текст"></applet> Ни один из браузеров альтернативный текст не отображает каким-либо образом. Альтернативный текст.
archive <applet code="URL" archive="URL файла с архивом"></applet> Не поддерживается браузерами Указывает путь и имя файла с архивом.
code <applet code="URL"></applet> Имя файла с расширением .class. Если расширение не указано (например: code="clock"), то браузер добавит расширение автоматически. Имя файла.
codebase <applet code="URL" codebase="URL"></applet>

Текущее время:

Используется для установки базового адреса, относительно которого будет задаваться значение атрибута code. Если значение codebase не указано, то в качестве адреса принимается адрес текущего документа.
height <applet code="URL" height="значение"></applet> Задаёт высоту апплета. Допускается использовать значения в пикселах или процентах. Если установлена процентная запись, то размеры апплета вычисляются относительно размера родительского элемента — контейнера, где находится тег <applet>.
hspace <applet code="URL" hspace="отступ по горизонтали"></applet> Горизонтальный отступ от апплета до окружающего контента.
vspace <applet code="URL" vspace="отступ по вертикали"></applet> Вертикальный отступ от апплета до окружающего контента.
width <applet code="URL" width="значение"></applet> Задаёт ширину апплета. Допускается использовать значения в пикселах или процентах. Если установлена процентная запись, то размеры апплета вычисляются относительно размера родительского элемента — контейнера, где находится тег <applet>. В случае отсутствия родительского контейнера, в его качестве выступает окно браузера. Иными словами, width="100%" означает, что апплет будет растянут на всю ширину веб-страницы.
