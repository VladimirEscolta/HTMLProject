Тег <video>

Описание

Добавляет, воспроизводит и управляет настройками видеоролика на веб-странице. Путь к файлу задается через атрибут src или вложенный тег <source>.

Атрибуты

autoplay <video autoplay="autoplay">...</video>

При наличии этого атрибута видео начинает воспроизводиться автоматически после загрузки страницы. Атрибут autoplay отменяет действие preload.
controls <video controls="controls">...</video>

Добавляет панель управления к видеоролику. Вид панели и ее содержимое зависит от браузера и может в себя включать: кнопка воспроизвести, пауза, перемотка, переход в полноэкранный режим; ползунок для изменения уровня громкости и др.
height <video controls="controls">...</video>

Атрибут height задает высоту области для воспроизведения видеоролика. Само видео меняет свои размеры в большую или меньшую сторону, чтобы вписаться в заданные рамки, но его пропорции при этом остаются прежними.
loop <video loop>...</video>

Зацикливает воспроизведение видео, оно повторяется каждый раз с начала после завершения.
poster <video poster="URL">...</video>

Указывает адрес картинки, которая будет отображаться, пока видео не доступно или не вопроизводится. Само изображение может быть в любом доступном формате: GIF, PNG, JPEG. Если атрибут poster не указан, браузер постарается отобразить первый кадр видео.
preload <audio preload="none | metadata | auto">...</audio>

Используется для загрузки видео вместе с загрузкой веб-страницы. Этот атрибут игнорируется, если установлен autoplay.

none
Не загружать аудиофайл.
metadata
Загрузить только служебную информацию (продолжительность звучания и др.).
auto
Загрузить аудиофайл целиком при загрузке страницы.
src <audio src="URL">...</audio>

Указывает путь к воспроизводимому видеоролику. Для этой же цели можно использовать тег <source>.
width <audio src="URL">...</audio>

Атрибут width задает ширину области для воспроизведения видеоролика. Само видео меняет свои размеры в большую или меньшую сторону, чтобы вписаться в указанную ширину, но его пропорции при этом не искажаются.
