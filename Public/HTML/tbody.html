Тег <tbody>

Описание

Элемент <tbody> предназначен для хранения одной или нескольких строк таблицы. Это позволяет создавать структурные блоки, к которым можно применять единое оформление через стили, а также управлять их видом через скрипты.

Допускается применять несколько тегов <tbody> внутри контейнера <table>. Доступны и другие виды группировок строк — <tfoot> и <thead>, ни один из них не должен перекрываться с элементом <tbody>.

Атрибуты

align <tbody align="left | center | right | justify">...</tbody >
Ячейка 1 Ячейка 2
Ячейка 3 Ячейка 4

Задает выравнивание по горизонтали содержимого строк ячеек, которые расположены внутри элемента <tbody>. Если требуется применить разное выравнивание для каждой ячейки, воспользуйтесь стилями или используйте атрибут align для тега <td> или <th>.

left
Выравнивание по левому краю.
center
Выравнивание по центру.
right
Выравнивание по правому краю.
justify
Выравнивание по ширине (одновременно по правому и левому краю).
bgcolor <tbody bgcolor="цвет">...</tbody >
Ячейка 1 Ячейка 2
Ячейка 3 Ячейка 4
Ячейка 5 Ячейка 6

Устанавливает цвет фона ячеек, расположенных в контейнере <tbody>.

Атрибут char заставляет выравниваться содержимое ячеек внутри <tbody> по заданному символу. Эта возможность может пригодиться для выравнивания колонки чисел по точке или запятой. Работает только в том случае, если значение атрибута align установлено как char.

Если атрибут align установлен как char, а для атрибута char в качестве значения задан символ, то с помощью атрибута charoff можно смещать содержимое ячейки влево или вправо относительно заданного символа. Так, charoff="3" означает, что браузер должен сместить содержимое ячейки на три символа вправо, а charoff="-3" — сместить на три символа влево.
valign <tbody valign="top | middle | bottom | baseline">...</tbody >
Ячейка 1 Ячейка 2
Ячейка 3 Ячейка 4
Ячейка 5 Ячейка 6
Ячейка 7 Ячейка 8

Устанавливает вертикальное выравнивание содержимого ячеек в контейнере <tbody>. По умолчанию контент ячейки располагается по ее вертикали в центре.

top
Выравнивание содержимого ячеек по верхнему краю строки.
middle
Выравнивание по середине.
bottom
Выравнивание по нижнему краю.
baseline
Выравнивание по базовой линии, при этом происходит привязка содержимого ячеек строки к одной линии.
