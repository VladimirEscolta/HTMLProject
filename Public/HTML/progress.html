Тег <progress>

Описание

Используется для отображения прогресса завершенности задачи. Изменение значения происходит через JavaScript.

Атрибуты

... <progress value="<число>" max="<число>">Текст</progress> Пожалуйста, подождите, фотографии загружаются. Загружено на 25%
value
Текущее значение прогресса.
max
Максимальное значение прогресса.
