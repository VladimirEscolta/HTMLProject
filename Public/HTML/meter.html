Тег <meter>

Описание

Используется для вывода значения в некотором известном диапазоне. Используется преимущественно для отображения числовых значений, например, количества результатов поиска, объема жидкости, давления и др.

Атрибуты

value <meter value="значение">Текст</meter> Процент зарядки смартфона

Устанавливает значение единицы измерения. Значение должно быть в диапазоне, задаваемым атрибутами min и max.
min <meter value="значение" min="минимальное значение">Текст</meter> Процент зарядки смартфона

Задает минимальный порог, который может достигать значение.
max <meter value="значение" max="максимальное значение">Текст</meter> Процент зарядки смартфона

Задает максимальный порог, который может достигать значение.
low <meter value="значение" low="низкое значение">Текст</meter> Процент зарядки смартфона

Определяет предел, при достижении которого значение считается низким.
high <meter value="значение" high="высокое значение">Текст</meter> Процент зарядки смартфона

Определяет предел, при достижении которого значение считается высоким.
optimum <meter value="значение" optimum="оптимальное значение">Текст</meter>

Определяет наилучшее или оптимальное значение.
