Тег <blink>

Описание

Устанавливает мигание текста.

Атрибуты

... <blink>Текст</blink>

Внимание!

Не тыкайте в него палкой, он не мертвый, просто прикидывается.

 Аналог CSS - text-decoration: blink. Этот тег не входит в спецификацию HTML и его наличие приведет к невалидному коду.
