Тег <li>

Описание

Тег <li> определяет отдельный элемент списка. Внешний тег <ul> или <ol> устанавливает тип списка — маркированный или нумерованный.

Атрибуты

type <li type="disc | circle | square">...</li>
<li type="A | a | I | i | 1">...</li>
  1. Привет
  2. Пока
  3. Ок
  • Первый
  • Второй
  • Третий

Устанавливает вид маркера. Тип маркера и его значение зависит от внешнего контейнера <ul> или <ol>.
value <li value="число">...</li>
  1. Молоко
  2. Сметана
  3. Йогурт

Атрибут value устанавливает номер, с которого будет начинаться список. value применяется только для нумерованных списков, когда тег <li> находится внутри контейнера <ol>. При этом не имеет значения, какой тип списка установлен с помощью type, атрибут value одинаково работает и с римскими и с арабскими числами.
